# EduProject-Hub
Overview  EduProject Hub is a responsive educational project management website designed for students and learners.
The platform allows users to explore available projects, create their own projects, and enroll in projects of interest.
**Features**
Responsive design for desktop and mobile devices
Project showcase section with project cards
Create Project form
Project Enrollment form
Sticky navigation bar
Mobile-friendly hamburger menu
Modern UI with CSS styling
Footer with contact information and quick links
**Technologies Used** HTML5,CSS3,Responsive Web Design,Flexbox,CSS Grid
